# Security Policy

## Reporting a Vulnerability

To report a security vulnerability, email **usdtb@proton.me**.

Do not open a public GitHub issue for security vulnerabilities.

We will acknowledge receipt within 48 hours and work with you to understand and resolve the issue before any public disclosure.

## Scope

- Backend API security issues
- Wallet/authentication bypass
- Payment flow manipulation
- Smart contract vulnerabilities (if applicable)

Thank you for helping keep USDBT secure.
